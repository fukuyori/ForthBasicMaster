# はじめに

本書は、FORTH言語を基礎から応用まで体系的に学ぶための入門書である。

FORTHは1970年代にチャールズ・H・ムーアによって生み出された言語であり、その設計思想は半世紀を経た今なお、プログラミングの本質について多くの示唆を与えてくれる。スタックベースのアーキテクチャ、逆ポーランド記法による演算、インタプリタとコンパイラの融合、そして「すべてがワードである」という一貫した世界観——これらの特徴は、現代の主流言語とは異なる視点からプログラミングを見つめ直す機会を提供してくれる。

本書の構成は、読者が段階的に理解を深められるよう設計した。前半では、スタック操作、条件分岐、繰り返し構造といった基本概念を丁寧に解説する。中盤では、変数・定数・ローカル変数といったデータ管理の手法や、CREATE/DOES>による動的構造の構築を扱う。後半では、浮動小数点演算、ファイル入出力、数値積分、行列演算といった実践的なトピックへと進み、最終的にはDSL（Domain Specific Language：領域特化言語）の構築へと至る。

各章には「補講」を設け、ボトムアップ設計、デバッグ技法、Thinking Forthの高速化哲学といった、コードを書く上での考え方についても触れた。FORTHを学ぶことは、単に新しい言語の文法を覚えることではない。それは、問題を小さな語彙に分解し、抽象化の層を積み重ね、最終的に自分だけの言語を創り上げるという、プログラミングの根源的な営みを体験することである。

本書では処理系としてGNU Projectのgforth（バージョン0.7.9以降）を使用する。gforthはオープンソースで広く普及しており、ほとんどのUNIX系環境ではsudo apt install gforthで導入できる。Windows、macOSでも利用可能であり、公式サイト（http://gforth.org/ ）から入手できる。

FORTHは「使いこなせば強力だが、最初は戸惑う」言語だと言われる。確かに、逆ポーランド記法やスタック操作は最初は慣れないかもしれない。しかし、それは数日で乗り越えられる壁であり、その先には「自分の思考を直接コードに変換できる」という、他の言語ではなかなか得られない体験が待っている。

本書が、読者のFORTHへの第一歩を支え、プログラミングの新たな視座を開く一助となれば幸いである。
